# TP1 ChompChamps

## Flujo de comandos

1) **make docker**
2) **make deps**
3) **make build**
4) **make run-demo**

### Ejecución del lanzador interactivo

Para usar el menú interactivo (antes `demo2`), ahora:

```
make play
```

Esto compila `bin/play` y lo ejecuta. Desde allí puedes configurar parámetros y lanzar `master`.
